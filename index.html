Aprendendo sobre links externos e internos

Links externos

Um link externo é quando necessário que a palavra se torne interativa e direcione o usuário para outra página. Ou seja a página que será direcionada não pertence ao site atual. Usando a tag A e dentro do HREF colocando a URL voçê realiza a ação de criar um link externo. Infelizmente desse modo o site é sobrescrito. Exemplo:

Navegação entre pastas

É necessário entender que para buscar uma pasta dentro do código Html principalmente no atributo HREF= digitamos ./ para encontrar elementos dentro da própia pasta, para encontrar elementos de uma pasta anterior devemos digitar ../

Links para downloads

Para fazer algo ser própicio para ser baixado da internet, é necessário a seguinte tag, atributo e valores; HREF=(o arquivo que será baixado)DOWNLOADS=(BAIXAR O ARQUIVO EM PDF/ZIP/HTML/ETC) TYPE="APPLICATION/(ZIP/PDF/HTML/ETC)"
